# local_syogi_dev
様々な対戦が可能な将棋アプリ

## 主な機能
* 認証系　　
* CRUD  
* ソケット通信  

## 言語
Kotlin

## アーキテクチャ
MVP + CleanArcitecture

## 主な使用技術
* socket.io
* Koin
* Retrofit
* OkHttp
* Gson
* coroutines
* ktLint
* firebase
* realm
